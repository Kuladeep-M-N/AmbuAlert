# AmbuAlert DT 🚑 

> **AI-Powered Digital Twin for Emergency Response Protocol**

AmbuAlert DT is a high-fidelity, military-grade mission control dashboard designed for emergency response coordination. Built for the IEEE TechX 2026 Live Ecosystem Demo, it provides real-time digital twin visualization and predictive analytics for ambulance dispatch and emergency management.

![AmbuAlert DT Banner](https://github.com/Kuladeep-M-N/AmbuAlert/raw/main/public/banner.png) *(Note: Placeholder for actual banner)*

## 🚀 Key Features

- **Mission Control Dashboard**: Real-time monitoring of emergency signals and ambulance status.
- **Predictive Analytics**: AI-driven route optimization and hospital load balancing.
- **Voice Triage Integration**: Automated triage processing via voice command protocols.
- **Global Impact Simulation**: Real-time data visualization of emergency response efficiency.
- **20s Emergency Protocol**: Rapid response simulation for critical event handling.

## 🛠️ Tech Stack

- **Frontend**: Vite + React
- **Styling**: Vanilla CSS (High-Performance Architecture)
- **Logic**: Javascript (Mission Critical Logic)
- **Design**: Cinematic / Military-Ops Aesthetics

## 📦 Installation

```bash
git clone https://github.com/Kuladeep-M-N/AmbuAlert.git
cd AmbuAlert
npm install
npm run dev
```
